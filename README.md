Данная утилита служит для сборки промпта, который можно использовать в ИИ для генерации юнит-тестов.

Основная идея утилиты:
1) На вход подаются: название класса и метода
2) На основе этих данных строим цепочку вызовов, а также собираем все ДТО и ENUM
3) После чего генерируем промпт

Как запустить проект:
Утилита представляет собой десктопное приложение, написанное на Swing. Для его запуска достаточно выполнить функцию main().
Откроется формочка, в которой нужно указать: класс, метод и путь до проекта.
После процесса анализа в том же окне отобразиться сгенерированный промпт.
