# Назначение утилиты  
Данная утилита предназначена для автоматической сборки промпта, который можно использовать в системах искусственного интеллекта для генерации юнит-тестов.  

## Основная идея работы  
1. На вход подаются название класса и метода.  
2. На основе этих данных утилита формирует цепочку вызовов и собирает все связанные DTO и ENUM.  
3. Затем на основе полученной информации автоматически генерируется промпт.  

## Как запустить проект  
Утилита представляет собой десктопное приложение, разработанное с использованием **Swing**.  
Для запуска достаточно выполнить функцию `main()`. После этого откроется окно, где необходимо указать:  
- название класса,  
- название метода,  
- путь к проекту.  

После анализа кода в этом же окне отобразится сгенерированный промпт.
